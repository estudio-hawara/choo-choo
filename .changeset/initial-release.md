Initial public release (`0.1.0`).

- `@choo-choo/core`: IR, manual builder, pure SVG renderer, `composeRule()` transform, shared `styles.css`. Zero runtime dependencies.
- `@choo-choo/parser-utils`: shared reader, tokenizer, specification, and error primitives for grammar parsers.
- `@choo-choo/parser-ebnf`: ISO/IEC 14977 EBNF parser.
- `@choo-choo/parser-antlr`: ANTLR4 grammar parser.
- `@choo-choo/parser-peg`: PEG (peggy / PEG.js dialect) parser.
- `@choo-choo/vanilla`: `<choo-choo>` custom element and imperative `mount()` helper.
- `@choo-choo/react`: SSR-safe React `<ChooChoo>` component.
- `@choo-choo/vue`: SSR-safe Vue 3 `<ChooChoo>` component.
- `@choo-choo/astro`: SSR-only Astro `<ChooChoo>` component, published source-only.
