# Contributing to choo-choo

The full contribution guide lives at [`docs/contributing.md`](./docs/contributing.md), alongside the rest of the project specs.

In short:

1. **Spec first.** Update `README.md` and/or `docs/**` to describe the change.
2. **Code.** Implement it.
3. **Tests.** Back it with Vitest tests.
4. **Changeset.** Run `pnpm changeset` and commit the generated file.
5. Open a pull request.

See [`docs/contributing.md`](./docs/contributing.md) for setup, commands, commit style, and where to add new grammar parsers or framework bindings. See [`docs/releasing.md`](./docs/releasing.md) for how versioning and publishing work.
