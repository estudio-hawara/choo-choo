Two walkthroughs from an empty directory to a railroad diagram rendered in the browser. Both use [`@choo-choo/vanilla`](../bindings/vanilla.md) plus [Vite](https://vitejs.dev) as a zero-config dev server.

## Prerequisites

- Node.js ≥ 20.
- `pnpm` (≥ 10). `npm` and `yarn` work too — swap the commands accordingly.

## First Steps

- **[Hello world](./hello-world.md)**: Build a minimal diagram by hand.
- **[Grammar example](./grammar.md)**: Parse an EBNF source and render a rule from it.
